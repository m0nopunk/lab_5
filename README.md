# Лабораторна робота №5. Робота з системою контролю версій Git

**Тема:** Використання гілок та віддалених репозиторіїв у системі Git

**Опис:** Практичне застосування механізмів розгалуження, злиття змін та роботи з віддаленими сховищами

**Мета:** Набути практичних навичок роботи з гілками, злиттям та віддаленими репозиторіями в Git

## Контрольні питання

1. **Яка різниця між Git та GitHub?**
   Git - система контролю версій, GitHub - хмарна платформа для зберігання Git-репозиторіїв.

2. **Для чого потрібен .gitignore?**
   Щоб Git не відстежував певні файли (тимчасові, конфіденційні, залежності).
